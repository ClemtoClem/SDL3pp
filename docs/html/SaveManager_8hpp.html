Detailed Description

Binary save/load for Polyadventure player progression.

Save format (little-endian, version-prefixed):

[4 bytes] magic = "PAVE" [1 byte] version = 1 [4 bytes] polypoints (int32) [4 bytes] hp (float) [4 bytes] maxHp (float) [N bytes] mapName (uint16 length + UTF-8 bytes, no NUL) [4 bytes] playerX (float) [4 bytes] playerY (float) [1 byte] playerDir (uint8, Direction enum)
